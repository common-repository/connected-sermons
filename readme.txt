## Description ##

### Easy to use Sermon Management ###


The easiest way to connect your message with your audience. Easily create, organize, and share sermons and videos.

* Add Preachers, Series, Topics, and Books.
* Attach images to Sermons.	
* Bible references integrated via Bib.ly
* Support for YouTube
* Support for FaceBook
* Support for MP3 (audio only sermon)
* Support for universal embed codes
* Support for self-hosted MP4s
* Quick support straight from the developers.
* Works with most templates.


### Shortcodes ###

* `[cs-sermons-list]` — This will list all sermons.
* `[cs-latest-sermon]` — (PRO) This will show the latest sermon as a single view.
* `[cs-sermons-filter]` - (PRO) This will display the filters for the list view.
* `[cs-series-list id=2]` - (PRO) This will display all sermons for a given series in a minimalistic view.

For the complete and latest information see our [documentation](https://church.agency/docs/).
